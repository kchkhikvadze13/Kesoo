Welcome to my demo!

Today, I will expalin how to advance your website by adding the dark mode option!

Light-on-dark color scheme—also called black mode, dark mode, dark theme or night mode—is a color scheme that uses light-colored text, icons, and graphical user interface elements on a dark background and is often discussed in terms of computer user interface design and web design.

Step 1:
First, Let's create a button in our index.html. At the and of the demo, by clicking this button the viewer will be able to toggle dark mode! 


                
                   button onclick="myFunction()">Switch to dark mode

                
                note: add button tag from both sides!

Step2:
Now we need to apply css to our code!

                
                      .dark-mode {
                        background-color: black;
                        color: white;
                      }

Step3:
Now it's time for magic to happen! let's add a little bit of Java Script

            
                function myFunction() {
                    var element = document.body;
                    element.classList.toggle("dark-mode");
                 }
                 
            
            
note: add this code under script tag!

Now try it by yourself!
